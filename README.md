# What is this?

Last updated May 12 2021

Hello person-who-has-found-their-way-here!


I have had acute tendonitis since 2015, which causes me a lot of pain while typing (and writing). As a statistician and programmer, I was worried that my career would be over if I couldn't type, but I took a break and I made some major adjustments to how I do my work and I've somehow managed to be productive for the last few years.


My setup has changed over the years, but currently I control my computer using my voice. I use talon, which is a program that uses a spoken syntax to transcribe and execute commands on your computer. It's pretty fantastic, but the learning curve is steep. talon is free, but I use it with a Blue Yeti desk microphone (~$150?) ~~Dragon ($300), a fancy microphone (FlexyMike + SpeechWare USB MultiAdapter = ~$300?)~~ and sometimes with an eye tracking device (tobii eye tracker ~$150). ~~You don't need Dragon to work with talon, but I tried to use it once without it and it was a disaster and I gave up quickly.~~  The talon voice recognition engine "Conformer" is amazing and so much better than Dragon ever was, even with my bananas accent. talon has a pretty great slack as well, so feel free to check that out to ask any questions you have! Most of the folks on there are amazing programmers, so there are many very impressive hacks that I don't even know about (I've seen some pretty impressive stuff for vim). talon is awesome because it's very customisable. This folder contains my personal customisations.

I also use vimium for Chrome (amazing!), and ~~RSIGuard~~ Stretchly to force me to take breaks even when I'm dictating. I'm also pretty bad at replying to e-mails these days. I've never had a problem with voice strain (I chug tea), but I've heard of folks who do.


Overall this is the best set up for me, I hope it helps you! I know how crushing it is to not be able to do your job (especially when you love your job), so I hope you find a sustainable set up for you. Adjusting to voice coding takes time, and has significant cognitive load, but it may be worth it! Good luck!
